# SOS-Tunisie Dashboard Platform

This is a full-stack application for the SOS-Tunisie child protection platform, featuring role-based dashboards for Mothers/Aunts/Educators, Psychologists, and Directors.

## Project Structure

### Frontend (React)
- `src/components/` - Reusable UI components
- `src/pages/` - Page components for different dashboards and auth
- `src/services/` - API service functions
- `src/utils/` - Utility functions

### Backend (Node.js)
- `src/controllers/` - Route handlers
- `src/models/` - Data models
- `src/routes/` - API routes
- `src/middleware/` - Authentication and other middleware
- `src/services/` - Business logic, AI integration, PDF generation
- `src/config/` - Configuration files

## Features
- User authentication with email verification
- Role-based access control
- Report creation with AI-powered PDF generation
- Prioritization system for psychologists
- Workflow management for child protection cases

## Setup
1. Install dependencies for frontend and backend
2. Configure database (MongoDB recommended)
3. Set up environment variables
4. Run the application
